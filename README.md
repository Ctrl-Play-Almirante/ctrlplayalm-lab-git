# 🎮 Ctrl+Play Almirante Tamandaré — Lab Open Source

> Repositório oficial do lab **"Contribuindo em um Projeto Open Source no GitHub"** da [Ctrl+Play Almirante Tamandaré](https://ctrlplay.com.br).

---

## 🚀 Sobre o Projeto

Este repositório foi criado para você praticar como funciona a contribuição em projetos open source 🌟

Aqui você vai aprender na prática:

- ✅ Como fazer um **fork** de um repositório
- ✅ Como **clonar** o projeto na sua máquina
- ✅ Como criar uma **branch** para a sua contribuição
- ✅ Como fazer um **commit** com suas alterações
- ✅ Como abrir um **Pull Request**

---

## 🎯 Como Contribuir

Existem **duas formas** de contribuir com este repositório:

### 1. 👤 Crie o seu perfil

Adicione um arquivo Markdown com o seu perfil na pasta `community/`. O arquivo deve ter o nome do seu usuário do GitHub.

**Exemplo:** `community/seu-usuario.md`

Veja o [perfil de exemplo](EXEMPLO.md) ou [perfil da prof](community/Caroline-Teixeira.md) para se inspirar!

### 2. 🌟 Adicione um projeto pessoal

No seu arquivo de perfil, você também pode listar projetos que você está desenvolvendo ou quer compartilhar com a turma.
